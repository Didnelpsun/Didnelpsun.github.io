使用 `|`来分隔不同的单元格，使用 `-`来分隔表头和其他行。`-`上面是表头，下面是表体，而如果没有`-`，就没有表头。

<span style="color:aqua">格式：</span>

```markdown
|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |
```

结果如下：

|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |

我们可以设置表格的对齐方式：  
-: 设置内容和标题栏居右对齐。  
:- 设置内容和标题栏居左对齐。  
:-: 设置内容和标题栏居中对齐。  

实例如下：

| 左对齐示例 | 右对齐示例 | 居中对齐示例 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
