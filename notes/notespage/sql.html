SQL

首先需要对这个sql图标有所说明，因为sql语言没有固定的图标，而我使用的是mysql且mysql的实现和标准sql语句的类似，对比sqlserver而言则与sql标准语言有一定的区别，所以我的sql图标就选择的是mysql的海豚图标，但是为了区别使用双海豚的变种图标

sql语句是一种标准，不同的数据库系统会有不同的使用方式，所以会在讲述完sql语句标准后讲述各种不同数据库的不同的语言标准，而且大部分数据库都会在这里讲述，而不会分为不同的模块来讲述。

且这里只会讲sql语法与使用的知识，而不会讲述数据库本身的知识，对于这方面请到其他的地方查看笔记。

SQL是一种用于操作数据库的语言。SQL是用于所有数据库的基本语言。不同数据库之间存在较小的语法更改，但基本的SQL语法基本保持不变。SQL是Structured Query Language 的简短缩写。它是非过程式的语言，也就是说它更像是一种指令，是让计算机去做 什么而得到想要的结果，而不关心它到底如何完成这个结果的，这就让程序员不用关系数据库的底层操作而更关注于数据库数据的操作。
